# BSO BlendShape Overwriter

## 概要
VRChat用のUnityアドオンです。  
口の形やサイズの改変により発生するリップシンク用シェイプキーの破綻を、アニメーションにてシェイプキーを上書きすることにより修正するためのアドオンです。

詳細なドキュメントは[こちら](https://kusumi-be.github.io/)
